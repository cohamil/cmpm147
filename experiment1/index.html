Experiment 1 - Getting Started

Description

Create an item generator that creates a random item name and random ability. Created using glitch.com and moved to this page.

Technical

The script uses a template text with a number of variables that are changed based on the iteration. For each of these variables, a random string is collected from an array of strings, which is then displayed in the text. This process is repeated until the story is created.

Reflection

This process was a great way for me to refresh myself on HTML, Javascript, and CSS. My main problem was formatting the text to appear nicely on the page, which I fixed by using Professor Modes' CSS code. I believe that I succeeded in my plan to create a random item generator, as the many options available can create very unique combinations.
